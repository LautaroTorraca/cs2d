- ClientHandler: Lee los comandos que manda el client (delegue a los handlers segun estapa).
        - LobbyHandler
        - GameLobbyHandler
        - InGameHandler

- Order: representa una order en la cola
        - OrderLobby: Devuelve el tipo de comando que se tiene que ejectuar
                                      [GAME LIST RESQUEST, JOIN GAME, CREATE GAME, DISCONNECT]

        - GameOrderLobby: Devuelve el tipo de comando que se tiene que ejectuar
                                      [SELECT_TEAM, READY, EXIT_LOBBY]

        - InGameOrder:  Devuelve el tipo de comando que se tiene que ejectuar
                                      [PLAYER_MOVEMENT, EXIT_GAME]

- Protocol: agarra una request, delega al protocolo correspondiente
                y la transforma en order

        - ServerLobbyProtocol: responde sus requests de protocol
        - ServerGameLobbyProtocol: responde sus requests de protocol
        - InGameProtocol: responde sus requests de protocol


Order -> protocol -> Handlers -> Servers
